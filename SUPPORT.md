# Support

This repository is maintained as an independent open-source project.

## Questions

Use GitHub Discussions when enabled. Otherwise, open a GitHub issue with a
minimal reproduction and sanitized data.

## Bugs

Use the bug-report template and include:

- package version or commit SHA,
- Node.js version,
- a minimal reproduction,
- expected and actual behavior,
- sanitized logs.

Do not include API keys, client secrets, access tokens, refresh tokens,
customer data, addresses, phone numbers or complete order payloads.

## Platform account problems

Account activation, application approval, permission scopes and marketplace
availability are controlled by the platform. Use the platform's official
support channel for account-level problems.
